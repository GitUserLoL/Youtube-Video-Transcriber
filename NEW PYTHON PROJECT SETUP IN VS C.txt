📁 NEW PYTHON PROJECT SETUP IN VS CODE
======================================

1. 📂 Open or Create a New Folder
   - File → Open Folder... → Choose or make your project folder

2. 💻 Open Terminal
   - Press Ctrl + ` (backtick) to open the terminal
   - Ensure you are inside the project directory

3. 🐍 Create a Virtual Environment
   - Run:
     python -m venv .venv

4. ⚡ Activate the Virtual Environment
   - Windows:
     .venv\Scripts\activate
   - macOS/Linux:
     source .venv/bin/activate

5. 📦 Install Required Packages
   - Example:
     pip install numpy pandas matplotlib

6. 🧊 Freeze Dependencies
   - Save installed packages to a file:
     pip freeze > requirements.txt

7. 🧠 (Optional) Set Python Interpreter in VS Code
   - Press Ctrl + Shift + P → "Python: Select Interpreter"
   - Choose the interpreter from `.venv`

8. ✅ Start Coding!
